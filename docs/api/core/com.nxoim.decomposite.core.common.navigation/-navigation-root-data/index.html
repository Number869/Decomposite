Creates the root of the app for back gesture handling, storing view models, and navigation controller instances.

ViewModelStore by default is wrapped into the default component context's instance keeper.

Initialize this outside of composition (application on JVM, setContent on Android, etc.).

This class is responsible for setting up the root context for navigation and state management within the application. It provides access to essential components like the ViewModelStore, NavControllerStore, and the default ComponentContext.

Each platform is expected to provide the default implementation of NavigationRootData.
